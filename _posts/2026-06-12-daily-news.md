## [Tolaria：用於 Markdown 知識庫管理的開源桌面應用程式](https://github.com/refactoringhq/tolaria) ⭐️ 6.0/10

refactoringhq 發布了 Tolaria，這是一個使用 TypeScript 編寫的開源桌面應用程式，用於在 macOS、Windows 和 Linux 上管理 Markdown 知識庫。 隨著個人知識管理（PKM）工具越來越重要，Tolaria 提供了一個專用的跨平台解決方案，利用輕量級的 Markdown 檔案來建立「第二大腦」並組織複雜資訊。 該應用程式支援多種使用案例，包括操作第二大腦、為 AI 建立公司文件上下文以及儲存程序；最近的更新包括主題匹配的應用程式圖示和多密庫類型可見性。

ossinsight · refactoringhq · 6月11日 23:30

**背景**: Markdown 是一種輕量級標記語言，允許使用者使用純文字撰寫格式化的文本，使其極具可攜性，非常適合建立個人知識庫（PKB）。Obsidian 和 Logseq 等工具是用於基於 Markdown 的知識管理熱門範例。Tolaria 旨在為此目的提供一個專用的桌面環境。
